# Product Overview

Mistral Worldwide Hackathon プロジェクト - 車両ナンバープレートのプライバシーを保護しつつ、ブロックチェーン上で車両のアイデンティティを管理するシステム。

## Core Capabilities

1. **ZK ベースの車両ID**: ナンバープレート番号をゼロ知識証明（Poseidon ハッシュ）でコミットメント化し、プライバシーを保護
2. **ERC-4337 ウォレット**: コミットメントから決定論的にアカウント抽象化ウォレットアドレスを生成
3. **音声会話インターフェース**: ElevenLabs を使った AI エージェントとの音声/テキスト対話
4. **MCP サーバー**: バックエンド API として機能するサービス

## Target Use Cases

- 車両のナンバープレートを公開せずにオンチェーンで車両アイデンティティを証明
- ゼロ知識証明を使った車両所有権の検証
- AI エージェントとの対話による操作支援

## Value Proposition

- **プライバシー保護**: 実際のナンバープレート番号をオンチェーンに一切公開しない
- **決定論的ウォレット**: 同じナンバープレートから常に同じウォレットアドレスを導出可能
- **ZK 証明**: ナンバープレートを明かさずに所有権を数学的に証明
