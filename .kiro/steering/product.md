# Product Overview

Mistral Worldwide Hackathon プロジェクト - 操作者の声のプライバシーを保護しつつ、ブロックチェーン上でアイデンティティを管理するシステム。

声の特徴量をゼロ知識証明により秘匿化しつつ、決定論的にERC4337規格準拠のスマートウォレットを生成するような仕組みを実現する。

声からウォレットを作成したり、資産を確認・操作する全ての工程はMCPサーバー越しに音声によって実現されるものとする。

## Core Capabilities

1. **ZK ベースの車両ID**: 操作者の声の特徴量をゼロ知識証明（Poseidon ハッシュ）でコミットメント化し、プライバシーを保護
2. **ERC-4337 ウォレット**: コミットメントから決定論的にアカウント抽象化ウォレットアドレスを生成
3. **音声会話インターフェース**: ElevenLabs を使った AI エージェントとの音声/テキスト対話
4. **MCP サーバー**: バックエンド API として機能するサービス

## Target Use Cases

- 操作者の声の特徴量を公開せずにオンチェーンで車両アイデンティティを証明
- ゼロ知識証明を使った車両所有権の検証
- AI エージェントとの対話による操作支援

## Value Proposition

- **プライバシー保護**: 実際の操作者の声の特徴量をオンチェーンに一切公開しない
- **決定論的ウォレット**: 同じ操作者の声の特徴量から常に同じウォレットアドレスを導出可能
- **ZK 証明**: 操作者の声の特徴量を明かさずに所有権を数学的に証明
